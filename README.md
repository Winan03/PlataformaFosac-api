# Control de Pasarelas de Pago

Este proyecto es un backend en Java que integra la pasarela de pago Stripe utilizando Spring Boot.

## Funcionalidades

- Creación de cargos mediante Stripe.
- Gestión de transacciones.

## Requisitos

- Java 17
- Maven
- Clave API de Stripe
